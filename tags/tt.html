<tt>

NÃO SUPORTADO EM HTML5.

A tag <tt>tag foi usada no HTML 4 para definir o tipo do texto

Exemplo

O que usar em vez da tag <tt>:

Considere o elemento <kbd> (para entrada do teclado), o elemento <var> (para variáveis), o elemento <code> (para código de computador), o elemento <samp> (para saída de computador) ou use CSS.

<p style="font-family:'Lucida Console', monospace">This text is monospace text.</p>
