<bdo>

BDO significa Substituição Bidirecional.

A tag <bdo> é usada para substituir a direção atual do texto.

Atributos

Atributo Valor Descrição
dir ltr
rtl		 Obrigatório. Especifica a direção do texto dentro do elemento <bdo>

Exemplo

Especifica a direção do texto:

<bdo dir="rtl">
  Este texto irá da direita para a esquerda.
</bdo>

Suporte ao navegador

A tag <bdo> é compatível com todos os principais navegadores.

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <bdo> com os seguintes valores padrão:

bdo {
  unicode-bidi: bidi-override;
}
