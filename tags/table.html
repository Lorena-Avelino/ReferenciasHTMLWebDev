<table>

A tag <table> exibe dados de maneira tabular e estruturada.

Uma tabela HTML consiste em um elemento <table> e um ou mais elementos <tr>, <th> e <td>. O elemento <tr> define uma linha da tabela, o elemento <th> define um cabeçalho de tabela e o elemento <td> define uma célula da tabela. Uma tabela HTML também pode incluir elementos <caption>, <colgroup>, <thead>, <tfoot> e <tbody>.

Exemplo

Uma tabela simples de HTML, contendo duas colunas e duas linhas:

<!DOCTYPE html>
  <html>
    <head>
     <style>

      table, th, td {
        border:1px solid black;
      }
    </style>
   </head>

<body>
   <h1>Elemento da Tabela</h1>
    <table>
    <tr>
     <th> Mês </th>
     <th> Economia </th>
    </tr>
    <tr>
     <th> Janeiro </th>
     <th> R$100 </th>
    </tr>
    <tr>
     <th> Fevereiro </th>
     <th> R$80 </th>
    </tr>
    </table>

</body>
</html>

Suporte ao navegador

A tag <table> é compatível com todos os principais navegadores.

Configurações padrão de CSS

table {
  display: table;
  border-collapse: separate;
  border-spacing: 2px;
  border-color: gray;
}
