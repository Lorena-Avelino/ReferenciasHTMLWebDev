<picture>

A tag <picture> oferece aos desenvolvedores web mais flexibilidade na especificação de recursos de imagem.

O uso mais comum do elemento <picture> será para direção de arte em designs responsivos. Em vez de ter uma imagem ampliada ou reduzida com base na largura da janela de visualização, várias imagens podem ser projetadas para preencher melhor a janela de visualização do navegador.

O elemento <picture> contém duas tags: uma ou mais tags <source> e uma tag <img>.

O navegador procurará o primeiro elemento <source> onde a consulta de mídia corresponde à largura da janela de visualização atual e, em seguida, exibirá a imagem adequada (especificada no atributo srcset). O elemento <img> é necessário como o último filho do elemento <picture>, como uma opção de fallback se nenhuma das tags de origem corresponder.

Dica: O elemento <picture> funciona de forma "semelhante" a <video> e <audio>. Você configura fontes diferentes, e a primeira fonte que atende às preferências é aquela que está sendo usada.

Exemplo

Desenhar um círculo:

<picture>
  <source media="(min-width:650px)" srcset="img_pink_flowers.jpg">
  <source media="(min-width:465px)" srcset="img_white_flower.jpg">
  <img src="img_orange_flowers.jpg" alt="Flowers" style="width:auto;">
</picture>

A tag <picture> é compatível com todos os principais navegadores.
