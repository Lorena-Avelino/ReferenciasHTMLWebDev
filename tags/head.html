<head>

A tag <head> é um contêiner para metadados (dados sobre dados) e é colocado entre a tag <html> e a <body> tag.

Os metadados normalmente definem o título do documento, o conjunto de caracteres, os estilos, os scripts e outras meta-informações.

Os seguintes elementos podem ir dentro do elemento <head>:

  • <tilte> (obrigatório)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>

Nota: Metadados são dados sobre o documento HTML. Os metadados não são exibidos.

Exemplo

Endereços de exemplos:

<address>
Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
</address>

Suporte ao navegador
