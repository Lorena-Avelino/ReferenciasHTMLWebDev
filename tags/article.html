<article>

A tag <article> especifica conteúdo independente e autocontido.

Um artigo deve fazer sentido por si só e deve ser possível distribuí-lo independentemente do resto do site.

Fontes potenciais para o elemento <article> :

  • Postagem de um fórum
  • Postagem de um blog
  • Notícia

Nota: Esta tag não renderiza nada de especial em um navegador, você tem que usar CSS para isso.

Exemplo

Artigos de exemplos:

<article>
<h2>Google Chrome</h2>
<p>Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!</p>
</article>

<article>
<h2>Mozilla Firefox</h2>
<p>Mozilla Firefox is an open-source web browser developed by Mozilla. Firefox has been the second most popular web browser since January, 2018.</p>
</article>

<article>
<h2>Microsoft Edge</h2>
<p>Microsoft Edge is a web browser developed by Microsoft, released in 2015. Microsoft Edge replaced Internet Explorer.</p>
</article>

Suporte ao navegador

A tag <article> é compatível com todos os principais navegadores.

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <article> com os seguintes valores padrão:

article {
  display: block;
}
