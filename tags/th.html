<th>

A tag <th> define uma célula de cabeçalho em uma tabela HTML.

Uma tabela HTML possui dois tipos de células:

  • Células de cabeçalho - contém informações de cabeçalho (criadas com o <th> elemento)
  • Células de dados - contém dados (criados com o elemento <td>)

O texto nos <th>elementos está em negrito e centralizado por padrão.

O texto nos elementos <td> é regular e alinhado à esquerda por padrão.

Exemplo

Uma tabela HTML simples com três linhas, duas células de cabeçalho e quatro células de dados:

<table>
  <tr>
   <th>Mês</th>
   <th>Saldo</th>
  </tr>
  <tr>
   <td>January</td>
   <td>$100</td>
  </tr>
  <tr>
   <td>February</td>
   <td>$80</td>
  </tr>
</table>

Suporte ao navegador

A tag <th> é compatível com todos os principais navegadores.

Atributos

Atributo Valor Descrição
abbr texto Especifica uma versão abreviada do conteúdo em uma célula de cabeçalho.
colspan number Especifica o número de colunas que uma célula de cabeçalho deve abranger.
headers header_id Especifica uma ou mais células de cabeçalho às quais uma célula está relacionada.
rowspan number Especifica o número de linhas que uma célula de cabeçalho deve abranger.
scope col, colgroup, row, rowgroup Especifica se uma célula de cabeçalho é um cabeçalho para uma coluna, linha ou grupo de colunas ou linhas
