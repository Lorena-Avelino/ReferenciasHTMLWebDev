<b>

A tag <b> tag especifica texto em negrito sem qualquer importância extra.

Nota: De acordo com a especificação HTML5, a tag <b> deve ser usada como ÚLTIMO recurso quando nenhuma outra tag for mais apropriada. A especificação afirma que os títulos devem ser denotados com as tags <h1> a <h6>, o texto enfatizado deve ser denotado com a tag <em>, o texto importante deve ser denotado com a tag <strong> e o texto marcado/destacado deve ser denotado com a tag <mark>.

Exemplo

Uma parte do texto sem negrito e outra com:

    <p> Este é um texto normal - <b>e este é um texto em negrito</b>.</p>

Suporte ao navegador

A tag <b> é compatível com todos os principais navegadores.

Configurações padrão de CSS

b {
  font-weight: bold;
}
