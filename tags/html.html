<html>

A tag <html> representa a raiz de um documento HTML.

A tag <html> é o contêiner para todos os outros elementos HTML (exceto para a tag <!DOCTYPE>).

Nota: Você deve sempre incluir o atributo lang dentro da tag <html>, para declarar o idioma da página da Web. Isto destina-se a ajudar os motores de busca e navegadores.

Atributos

Atributo Valor Descrição
xmlns http://www.w3.org/1999/xhtml Especifica o atributo de namespace XML (se você precisar que seu conteúdo esteja em conformidade com XHTML)

Exemplo

<!DOCTYPE html>
<html lang="en">
<head>
 <title>Title of the document</title>
</head>
<body>
 <h1>This is a heading</h1>
 <p>This is a paragraph.</p>
</body>
</html>

Suporte ao navegador

A tag <article> é compatível com todos os principais navegadores.

Configurações padrão de CSS

html {
  display: block;
}

html:focus {
  outline: none;
}
