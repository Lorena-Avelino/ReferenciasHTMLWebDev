<h1>-<h6>

As tags <h1> a <h6> são usadas para definir cabeçalhos HTML.

<h1> define o título mais importante. <h6> define o título menos importante.

Nota: Use apenas um <h1> por página - isso deve representar o título/assunto principal de toda a página. Além disso, não pule os níveis de título – comece com <h1> , depois use <h2> e assim por diante.

Dica: Use elementos <h1> a <h6> apenas para títulos. Não os use apenas para deixar o texto em negrito ou grande. Use outras tags para isso.

Exemplo

Exemplos de cada nivel da tag 'h':

<!DOCTYPE html>
  <html>
<body>
   <h1> Este é o h1! </h1>
   <h1> Este é o h2! </h1>
   <h1> Este é o h3! </h1>
   <h1> Este é o h4! </h1>
   <h1> Este é o h5! </h1>
   <h1> Este é o h6! </h1>

</body>
</html>

Suporte ao navegador

A tag <h1> ao <h6> são compatíveis com todos os principais navegadores.

Configurações padrão de CSS

h1 {
  display: block;
  font-size: 2em;
  margin-top: 0.67em;
  margin-bottom: 0.67em;
  margin-left: 0;
  margin-right: 0;
  font-weight: bold;
}
