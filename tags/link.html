<link>

A tag <link> define o relacionamento entre o documento atual e um recurso externo..

A tag <link> é mais frequentemente usada para vincular folhas de estilo externas ou para adicionar um favicon ao seu site.

O elemento <link> é um elemento vazio, contém apenas atributos.

Atributos

Atributo Valor Descrição
crossorigin anonymous
use-credentials		 Especifica como o elemento lida com solicitações de origem cruzada.
href URL Especifica a localização do documento vinculado.
hreflang language_code Especifica o idioma do texto no documento vinculado.
media media_query Especifica em qual dispositivo o documento vinculado será exibido.
referrerpolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
unsafe-url		 Especifica qual referenciador usar ao buscar o recurso.
rel alternate
author
dns-prefetch
help
icon
license
license
next
pingback
preconnect
prefetch
preload
prerender
prev
search
stylesheet		 Obrigatório. Especifica a relação entre o documento atual e o documento vinculado.
sizes HeigthxWidth
any		 Especifica o tamanho do recurso vinculado. Somente para rel="icon".
title Define uma folha de estilo preferida ou alternativa.
type media_type Especifica o tipo de mídia do documento vinculado.

Exemplo

Uma tabela simples de HTML, contendo duas colunas e duas linhas:

<!DOCTYPE html>
  <html>
    <head>
     <link rel="stylesheet' href="styles.css">
   </head>

<body>
   <h1>Hello World!</h1>
    <h2>Estou formatada com o linked style sheet.</h2>     <p>Eu também!!.</p>
</body>
</html>

Suporte ao navegador

A tag <link> é compatível com todos os principais navegadores.

Configurações padrão de CSS

link {
  display: none;
}
