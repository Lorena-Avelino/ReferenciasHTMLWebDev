<strong>

A tag <strong> é usada para definir textos de grande importância. O conteúdo interno normalmente é exibido em negrito.

Dica: Use a tag <b> para especificar texto em negrito sem qualquer importância extra!

Exemplo

Define um texto importante em um documento:

<strong>Este é um texto importante!</strong>

Suporte ao navegador

A tag <strong> é compatível com todos os principais navegadores.

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <strong> com os seguintes valores padrão:

strong {
  font-weight: bold;
}
